# 🧪 Tests automatisés de site web avec Selenium

Ce dépôt est destiné à la réalisation de **tests automatisés de sites web** à l'aide de **[Selenium](https://www.selenium.dev/)**, en utilisant Python et le framework `unittest`.

## 📂 Structure du projet

selenuim_tests/
│
├── test/
│ └── test_chargement_siteweb.py # Script principal de test (connexion, upload, etc.)
│
├── requirements.txt # Dépendances Python
└── README.md # Ce fichier


## 🛠️ Prérequis

- Python 3.x installé
- [Google Chrome](https://www.google.com/chrome/) installé
- [ChromeDriver](https://sites.google.com/a/chromium.org/chromedriver/) compatible avec la version de Chrome
- Les dépendances listées dans `requirements.txt`

## ⚙️ Installation

```bash
pip install -r requirements.txt
